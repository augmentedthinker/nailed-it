# Nailed It

A private, joyful social space for friends to share fresh nail sets, exchange inspiration, and hype each other up.

This repository contains the first mobile-first landing page for the product experiment.

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The site is a Vite + React application ready for Vercel deployment. Supabase will be introduced when the first social feature requires persisted data.
